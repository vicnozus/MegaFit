# MegaFit

## como rodar?
    abra o terminal e escreva o caminho da pasta
    se ja tiver o caminho, use uma dessas formas:
    
# npm start
    ele vai rodar o projeto nessa pagina: http://localhost:3000     funciona bem para apresentar

# npm run dev
    use esse comando para programar, ele atualiza o projeto sem a necessidade de rodar novamente.
